## 네오솔루션 VAN 서비스

네오솔루션은 전국 가맹점에 안정적이고 빠른 신용카드 결제 승인 서비스를 제공합니다.

### 주요 특징

- **빠른 승인 속도** — 평균 1.5초 이내 승인 완료
- **높은 안정성** — 이중화 네트워크로 99.9% 가동률 보장
- **전 카드사 지원** — 신한, 삼성, KB국민, 현대, 롯데, 하나, BC, NH농협 등 전 카드사 연동
- **합리적 수수료** — 업종별 최적화된 VAN 수수료 체계

### VAN이란?

VAN(Value Added Network)은 가맹점의 카드 결제 단말기와 카드사 사이에서 결제 승인 데이터를 안전하게 중계하는 부가가치통신 서비스입니다.

```
고객 카드 제시 → POS 단말기 → [VAN 네트워크] → 카드사 승인 → 결제 완료
```

### 가맹점 혜택

| 혜택 | 내용 |
|------|------|
| 무료 단말기 설치 | POS 도입 시 VAN 단말기 무상 제공 |
| 24시간 고객센터 | 결제 장애 발생 시 즉시 대응 |
| 매출 조회 서비스 | 일별/월별 카드 매출 실시간 확인 |
| 정산 관리 | 카드사별 정산 내역 통합 관리 |

### 지원 결제 수단

- 신용카드 / 체크카드
- 간편결제 (삼성페이, 카카오페이, 네이버페이, 제로페이)
- 현금영수증 발급
- 포인트 결제
