## 유통/소매 전용 POS 시스템

소매 유통 매장에 최적화된 POS 시스템으로, 바코드 스캔부터 재고 관리, 매출 분석까지 매장 운영에 필요한 모든 기능을 제공합니다.

### 주요 기능

#### 판매 관리
- 바코드 스캔 & 수기 입력 지원
- 복합 결제 (카드 + 현금 분할)
- 할인/쿠폰/포인트 적용
- 반품/교환 처리

#### 재고 관리
- 실시간 재고 현황 파악
- 자동 발주 알림 (최소 재고량 설정)
- 입출고 이력 추적
- 유통기한 관리

#### 매출 분석
- 일별/주별/월별 매출 리포트
- 상품별 판매 순위
- 시간대별 매출 패턴
- 객단가 분석

#### 회원 관리
- 포인트 적립 & 사용
- 회원 등급별 할인
- 구매 이력 조회
- SMS 마케팅 연동

### 적용 업종

- 편의점 / 슈퍼마켓
- 생활용품점
- 문구/완구점
- 화장품/드럭스토어
- 의류/잡화점

### 하드웨어 사양

| 구성 | 사양 |
|------|------|
| 본체 | 15" 터치스크린 일체형 |
| OS | Windows / Android 선택 |
| 프린터 | 80mm 열전사 (자동 커터) |
| 바코드 리더 | 1D/2D 겸용 |
| 고객 표시기 | 7" 서브 디스플레이 |
| 카드 리더 | IC/MS/NFC 겸용 |
