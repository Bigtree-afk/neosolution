## 외식업 전용 POS 시스템

일반음식점, 카페, 프랜차이즈까지 — 외식업의 특성을 반영한 전용 POS로 주문부터 결제, 주방 관리까지 하나로 통합합니다.

### 주요 기능

#### 주문 관리
- 테이블 배치도 기반 주문 입력
- 메뉴 카테고리별 빠른 선택
- 옵션/추가 토핑 설정
- 합석/분리/이동 처리

#### 주방 연동 (KDS)
- 주방 디스플레이 시스템(KDS) 연동
- 주문 전표 자동 출력
- 조리 완료 알림
- 코스별 순차 출력

#### 배달 통합
- 배민/요기요/쿠팡이츠 주문 자동 연동
- 배달/포장/홀 매출 분리 집계
- 배달비/수수료 자동 계산
- 라이더 호출 연동

#### 매출 분석
- 메뉴별 판매 분석
- 시간대별 좌석 회전율
- 배달 vs 홀 비중 분석
- 식재료 원가율 추적

### 적용 업종

- 일반음식점 / 한식/중식/일식/양식
- 카페 / 베이커리
- 패스트푸드 / 분식
- 프랜차이즈 (다매장 통합 관리)
- 술집 / 호프 / 바

### 부가 서비스

- **테이블 오더** — QR 코드 스캔으로 고객 직접 주문
- **키오스크 연동** — 무인 주문 시스템 통합
- **예약 관리** — 전화/온라인 예약 통합 관리
- **직원 근태** — 출퇴근 기록 & 급여 연동
