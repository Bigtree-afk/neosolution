## 식자재마트 전용 POS 시스템

식자재 유통의 특수성을 완벽히 반영한 전용 POS입니다. B2B 거래처 관리, 대량 주문 처리, 세금계산서 발행까지 식자재 도매/소매 매장에 필요한 모든 기능을 지원합니다.

### 주요 기능

#### 거래처(B2B) 관리
- 거래처별 외상 매출 관리
- 거래처 등급별 단가 차등 적용
- 외상 잔액 실시간 조회
- 거래처별 구매 이력 분석

#### 세금계산서 / 계산서
- 전자세금계산서 자동 발행 (국세청 연동)
- 거래명세서 출력
- 월말 일괄 정산
- 부가세 신고 자료 자동 생성

#### 상품 관리
- 중량 단위 판매 (kg, 근, 박스)
- 시세 변동 상품 단가 즉시 반영
- 유통기한 / 선입선출 관리
- 대분류/중분류/소분류 체계

#### 대량 주문 처리
- 전화/팩스 주문 일괄 입력
- 배송 루트별 주문 정리
- 출고 전표 & 거래명세서 동시 출력
- 차량별 배송 스케줄 관리

### 적용 업종

- 식자재마트 / 식자재 도매
- 농수산물 도매
- 축산물 전문점
- 공산품 도매
- B2B + B2C 복합 매장

### 식자재 특화 기능

| 기능 | 설명 |
|------|------|
| 중량 판매 | 저울 연동, kg/근/박스 단위 자동 계산 |
| 시세 관리 | 일별 시세 변동 상품 단가 즉시 업데이트 |
| 외상 관리 | 거래처별 외상 한도 설정 & 잔액 알림 |
| 배송 관리 | 루트별 배송 스케줄 & 출고 전표 |
| 세금계산서 | 전자세금계산서 자동 발행 & 국세청 전송 |
